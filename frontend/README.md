# FinBrain - Frontend

Frontend da plataforma FinBrain.

## 🚀 Como rodar

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev
```

Acesse `http://localhost:3000`

## 📁 Estrutura

```
src/
├── app/
│   ├── login/       # Página de login
│   ├── register/    # Página de registro
│   ├── confirmar/   # Confirmação de email
│   └── dashboard/   # Dashboard
├── services/
│   └── api.ts       # Comunicação com backend
└── public/
    └── images/      # Imagens da aplicação
```

## 🔧 Configuração

O backend deve estar rodando em `http://localhost:8080`

## 📦 Tecnologias

- Next.js 16
- TypeScript
- Tailwind CSS

## 🎨 Cores

- Roxo: `#6C47E8`
- Branco: `#FFFFFF`

## 📡 API

| Endpoint | Função |
|----------|--------|
| `/api/auth/register` | Registrar |
| `/api/auth/login` | Login |
| `/api/auth/confirmar` | Confirmar email |

## 🐛 Problemas comuns

**Erro de conexão**: Verifique se o backend está rodando na porta 8080

**Link do projeto**: [http://localhost:3000](http://localhost:3000)

Desenvolvido por FinBrain 💜
